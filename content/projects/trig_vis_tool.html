Trigger Monitoring Visualization Tool

A visualization tool with dashboarding capabilities to monitor the Trigger performance, powered by Jupyter notebooks, Python packaged and integrated in the cloud.

Jun 2019 - Jul 2022 Santiago Noacco

The Trigger Team is responsible for the early & fast selection of events in the detector, thus its efficiency is crucial to store high quality events for offline treatment. This Tool aimed to check the performance aspects of changes in the Trigger Software between releases.

ATLAS has a custom framework and cloud service which has support for Jupyter-Notebooks like scripts. I developed a Python package with tailored methods for interactive Search & Visualization on the notebooks, using the cloud access to the database.

As a result it allowed to quickly & easily explore the content of the DB, monitor the key aspects (metrics) of the Trigger performance: efficiency & resolution; as well as being readily available to distribute among teams.
