Inspector Tool

A Python package to distribute insights across data teams.

Dec 2021 - Mar 2022 Santiago Noacco

Powerful insight happens when knowledge is transfer across teams and it uplifts their products, performance and so on. The Media Buyer Teams where struggling to align methods, tools and rituals; plus we needed a way to share & distribute our tools without friction. Thus I end-to-end developed a Python package to fill that need, which I then deployed to Azure cloud for the teams to install through PIP.

This package was powered by Python & libraries such as: PySpark, Plotly, Pandas, SciPy and Numpy. It had embedded a number of visualization tools and analysis, for example to show the top performing campaigns given an input criteria. I also served as a flexible dashboard tool, where the Media Buyers where able to monitor aspects of the campaigns performance at a quick glance.

Four different Data Teams relay on this package to deliver their results.
